Technical Blog - Javascript Basic

By Sam Manongga | Post Date: March 14 2017

How does JavaScript compare to HTML and CSS?

Javascript is the third part that makes up a website (at the moment). HTML is a markup language which give website content some structure. CSS is the style sheet language improve website’s presentation. Javascript can serves interactivity for the end user. What I found Javascript differ from the other two is that it can store values, create objects, setup re-usable code (function) and handle logic … I’m sure I’m just scratching the surface.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

Control Flow - check to see if condition meets the criteria.
Before you can get a loan from a bank, you will need to convince the bank that you are a trustworthy person who is able to pay the loan back. At the bank, different departments will check your application and check off if individual criteria is met.

Loops - doing same thing until it meets goal.
A swimmer who is doing freestyle for 1000 meter in a 50 meter pool. The swimmer will do the same movement where after each stroke he will come up for breath. At every 50 meter, he will flip and swim the other direction until he reached 1000 meter.

Explain the difference between accessing data from arrays and objects.

Accessing data from arrays

To access array’s data, you will need to use its number position. The position start at 0.
var myArray = [“Spiderman” , “Thor” , “Captain America”]

In order to call Captain America, it will we will use myArray[2]

Accessing data from Object

With Object, you will use the name of the data label you want to access.

-

var person = {firstName: “Sam”, lastName: “Manongga”};

In order to access first name value - “Sam”, we will use … person.firstName

Explain what functions are and why they are useful.

Functions in Javascript allow where you group a set of code together and you to re-use grouped code in different parts of your program. This will make your code more manageable. Example: In case there is an error you can do faster debugging by pinpointing the faulty function instead of reading line by line.
